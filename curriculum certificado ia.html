Proyectos

  • Proyecto personal. Análisis de Seguridad Wi-Fi Doméstica
  • Este proyecto analizó la seguridad de una red Wi-Fi doméstica para identificar vulnerabilidades y recomendar mejoras. Utilizando la herramienta nmap, se escaneó la red para detectar dispositivos conectados y analizar la configuración del router (cifrado, contraseña, firewall). Se realizaron pruebas básicas de seguridad, como intentos de conexión con contraseñas débiles. Los resultados revelaron vulnerabilidades como una contraseña débil y un nombre de red visible. Se recomendó una contraseña más robusta, ocultar el SSID, activar el firewall y actualizar el firmware del router. El proyecto demostró la importancia de una configuración segura de la red Wi-Fi doméstica y la aplicación de mejores prácticas de seguridad. Se utilizó la herramienta gratuita nmap.
